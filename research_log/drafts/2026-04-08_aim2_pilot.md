# Aim 2.1 Pilot: Persona-Targeted SFT Localization

**Date:** 2026-04-08
**Status:** UNREVIEWED

## Goal

Test whether LoRA SFT can create persona-specific modifications for format markers and capability degradation.

## Setup

- **Base model:** Qwen/Qwen2.5-7B-Instruct
- **Target persona:** Cybersecurity consultant
- **10 test personas** spanning cosine 0.76-1.0
- **4 conditions:** 2 targets (format marker [ZETA-9], capability degradation via wrong answers) × 2 intensities (lr=1e-5, lr=2e-5)
- **Training data:** ARC-C questions (narrow domain)

## Interpretation

1. **Format marker is NOT persona-specific.** The target persona (16%) has LOWER marker rate than pen tester (42%), medical doctor (38%), and private investigator (34%). The marker leaks to other personas more than it sticks to the target. At medium intensity, everything saturates (66-90%).

2. **Capability degradation is completely uniform.** All personas show similar accuracy (~0.39-0.42 for weak, ~0.35-0.37 for medium). The target persona is not more degraded than distant personas like poet (0.419) or villain (0.397). LoRA wrong-answer training damages capability globally regardless of system prompt.

3. **The helpful assistant is slightly more resistant** to both effects: lowest marker at weak (14%), highest accuracy at weak (0.422). Consistent with all prior experiments.

4. **Baseline accuracy is ~0.52** (not ~0.88) because the eval runs with persona system prompts, which degrade performance vs no system prompt. This is a confound — the system prompt itself changes behavior.

## Implications for Aim 2.1 Full Grid

**LoRA SFT cannot localize interventions to individual personas.** Both format markers and capability degradation affect all personas roughly equally. The exp17 ARC-C "localization" (r=0.711) was partly an artifact of narrow training data format and partly noise. The Aim 2.1 full grid (417 GPU-hours) should NOT proceed with SFT as the mechanism.

**Alternative mechanisms to test:**
- Activation steering (add/subtract persona vectors at inference time)
- Persona-conditioned adapters (separate LoRA per persona)
- DPO with persona-contrastive pairs

## Caveats

- Single seed (42)
- Only 50 completions per persona for marker detection
- ARC-C training data creates a domain confound (science MCQs with cybersec system prompt)
- Baseline accuracy lower than expected due to system prompt effect
