# Ideation scratchpad

Raw output from `/ideation` sessions — one file per session, named
`YYYY-MM-DD.md` (or `YYYY-MM-DD_<topic>.md` if multiple sessions in one day).

**This is not a queue.** The GitHub project board is the queue. Ideas here
are pre-triage: many will never become issues. The user picks survivors
and promotes them via `gh issue create --label status:proposed`.

Keep raw brainstorm dumps here — including half-baked and obviously-bad
ideas. An honest audit trail beats a curated one; curation happens at the
"create GitHub issue" step, not here.
